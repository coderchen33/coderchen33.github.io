html快速上手

1. html简介

1.1什么是html

HTML（Hyper Text Markup Language）称为超文本标记语言。它不是一种编程语言，而是一种标记语言。它使用一系列标签来描述网页。这些标签经过浏览器解析后展示给用户。

1.2 html基本结构

2. 基本标签

  • 标题标签 <h1></h1> … <h6></h6>
  • 段落标签 <p></p>
  • 换行标签 <br>
  • 水平线标签 <hr>
  • 字体样式标签 加粗<stronger></stronger> 或者<em></em>
  • 特殊符号 ： 空格 &nbsp; 大于号 &gt; 小于号 &lt; 版权所有&copy;

3. 图像标签和超链接标签

  • 图像标签 <img src="" alt="" >
    • src 必填，表示文件路径
    • alt 必填，表示图片加载不出来时显示的文字
    • title 选填，鼠标悬停到图片上时显示的文字
  • 超链接标签 <a href=""></a>
    • href：必填，表示跳转的地址
    • target： 选填，表示在哪个窗口打开跳转的页面.
      _self :   不写也默认是当前页面
_blank：  在新标签页中打开
    • 锚链接参考下面的例子，通常用来进行返回顶部、返回底部等。

    • 跨行使用rowspan

  • 视频和音频<video src=""></video>

7. 表单标签

表单标签也是和我们后台打交道最多的地方，所以一定要重点学习。

表单标签的使用方式

  • 表单标签 <form></form>
    • action： 表单提交的位置，可以是网页或者是一个请求处理地址
    • method: get方式拼接在url后面 不安全高效
      post方式  传输大文件  安全

因为表单最终是要提交到后台，所以下面的这些标签都应该在form标签内。

  • input标签<input type=""> type表示输入框的类型

    • text 普通文本
    • password 密码
    • radio 单选框 注意：只有name相同的选项才能单选，否则不同name下可以多选
    • checkbox 多选框 注意：name相同的组内多选会作为数组提交

    重点：上面的类型中在和后台交互时 name为上传的键，value为上传的值

    • 按钮 可以自己绑定事件进行处理
      • button 普通按钮
      • image 图片按钮
      • submit 提交按钮
      • reset 重置按钮
    • 文件 file

  • 下拉框 <select name="" id=""> <option value=""></option> </select>

  • 文本域 <textarea></textarea>
