7.操作BOM对象

BOM (Browser Object Model) 是我们的浏览器对象模型。简单的说就是提供给了我们独立于页面内容，而与浏览器窗口进行交互的对象。

window对象(重要) 表示浏览器的窗口，我们可以通过window对象来查看移动调整浏览器窗口。

另外要注意的是，我们上面也提到过window对象是js中的顶级对象，所有定义在全局作用域中的变量、函数都会变成window对象的属性和方法，在调用的时候可以省略window。

screen对象 包含用户屏幕的信息

8.DOM对象

DOM（Document Object Model), 文档对象模型。当网页被加载时，浏览器会创建页面的文档对象模型，也就是我们平时说的HTML的DOM树。

我们js的核心也就是可以创建动态的DOM树。核心：

  • 更新： 更新DOM节点
  • 删除： 删除DOM节点

10.JQuery

什么是Jquery？

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

jQuery是一个快速，小型且功能丰富的JavaScript库。 通过易于使用的API（可在多种浏览器中使用），它使HTML文档的遍历和操作，事件处理，动画和Ajax等事情变得更加简单。 兼具多功能性和可扩展性，jQuery改变了数百万人编写JavaScript的方式。

简单一句话，JQuery是一个封装了大量原生js代码的js库，从而简化我们js代码的编写。
