1. js简介

1.1 什么是javascript

JavaScript（简称“JS”） 是一种即时编译型的高级编程语言。它是作为开发Web页面的脚本语言而出名的。JavaScript 基于原型编程、多范式的动态脚本语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。

  • HTML 定义网页的内容
  • CSS 规定网页的布局
  • JavaScript 对网页行为进行编程

使用js我们就能够和网页进行交互，比如弹窗、改变html属性等等操作。

3.3 变量的作用域

首先，回想一下java中都有哪些变量：

  • 局部变量(方法内定义和使用)
  • 全局变量(类内、没有static修饰，全类使用)
  • 类变量（类内、使用static修饰， 类中static修饰的方法使用）

那我们js中定义的变量有作用域吗？ 答案是肯定的。

方法中定义的变量类似于java中的局部变量 (不能被方法外使用) 非要使用需要js中的闭包

在方法中使用var定义一个变量，首先在方法中可是使用是没问题的。但是在方法外使用，就会在控制台报错x is not defined变量未定义。

或者我们直接使用严格检查模式use strict也会直接看到外面的x被标红

内部方法可以调用外部方法的成员，反之则不行

内部方法和外部方法变量同名时，内部方法优先使用内部变量的值 (和Java相同)

全局变量和 java中一样 定义之后可以被所有函数使用

实际上，js只有一个全局作用域window对象，任何全局变量(包括函数)都会绑定到window对象下。假设没有在函数范围内找到，就会向外查找，如果在全局作用域中都没有找到，就会报错RefrenceError。

这样，就会产生一个问题，如果多个人合作开发时，不同js文件使用了相同的全局变量，就会产生冲突，解决方法（也是规范）就是把自己的代码全部放入自己定义的唯一空间名字中，降低全局命名冲突的问题（这也是jQuery的实现方法）

const、let、var

  • const定义的变量不可以修改，而且必须初始化。
  • var定义的变量可以修改，如果不初始化会输出undefined，不会报错。
  • let是块级作用域，函数内部使用let定义后，对函数外部无影响。

JSON对象

首先，什么是JSON？？？

JSON(JavaScript Object Notation) 翻译过来就是 JavaScript 对象标记法。它是一种轻量级的数据存储和交换格式。易于人阅读和编写。同时也易于机器解析和生成，并有效地提升网络传输效率。 之后我们的前后端交互使用的数据都是JSON格式的。

JSON的格式：

  • 对象都在{}中
  • 数组都在[]中
  • 数据使用键值对 key:value表示
  • 数据间用,隔开

JSON字符串和js对象的对比：

JSON交换数据

  • 当数据在浏览器与服务器之间进行交换时，这些数据只能是文本。

  • JSON 属于文本，并且我们能够把任何 JavaScript 对象转换为 JSON，然后将 JSON 发送到服务器。

  • 我们也能把从服务器接收到的任何 JSON 转换为 JavaScript 对象。

  • 以这样的方式，我们能够把数据作为 JavaScript 对象来处理，无需复杂的解析和转译。

发送数据：将js对象转换为JSON串

接收数据： 将JSON串转为js对象

8. jQuery

什么是jQuery？

下面是jQuery官网的介绍：

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

jQuery是一个快速，小型且功能丰富的JavaScript库。 通过易于使用的API（可在多种浏览器中使用），它使HTML文档的遍历和操作，事件处理，动画和Ajax等事情变得更加简单。 兼具多功能性和可扩展性，jQuery改变了数百万人编写JavaScript的方式。

简单一句话，jQuery是一个封装了大量原生js代码的js库，从而简化我们js代码的编写。
